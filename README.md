# 📱 To-Do List App – SwiftUI

Ứng dụng quản lý công việc cá nhân viết bằng SwiftUI, kiến trúc MVVM.  
Các tính năng:

✅ Tạo – sửa – xoá task  
✅ Gắn mô tả, thời gian  
✅ Đính kèm ảnh, vị trí  
✅ Ghi chú hoàn thành  
✅ Nhóm task theo ngày  
✅ Chọn nhiều task → xoá nhanh

---

## 🛠 Công nghệ

- SwiftUI + MVVM  
- UserDefaults + Codable  
- PhotosUI + CoreLocation  
- List + Section + sheet()
